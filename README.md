# Space Chess

Пошаговая тактическая PvP («космические шахматы») на React + TypeScript + Vite.

## Что уже есть

- Главное меню: **Играть** / **Армии** / **Настройки**
- Сборка армий: бюджет **100**, до **16** слотов, фракции **Космодесант** и **Плутоны** + нейтралы
- Режим **Классический**:
  - **Онлайн** — пока недоступен
  - **Оффлайн** — два игрока на одном устройстве, ходы по очереди
- Поле **16×16**, спавн армий с 4-й клетки, бой и победа по командирам

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```
