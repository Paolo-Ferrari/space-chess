## 1. Цель документа

Описать циклы игрока: от запуска клиента до реванша — без изобретения новых механик.

## 2. Утверждённые петли (MVP)

### Core loop (матч)

Выбрать фигуру → выполнить одно действие (ход / атака / способность) → ход сопернику → повтор до победы/сдачи.

### Session loop

Меню → Играть → выбор режима → (локально: вторая вкладка / ожидание соперника) → матч → результат → меню / реванш (если появится).

### Meta loop (вне матча)

Меню → Армии → сборка под героя/фракцию (бюджет, слоты) → сохранение → использование в матче.

### App shell

Экраны верхнего уровня: **Играть** / **Армии** / **Настройки** (см. UI Flow).
