## 1. Цель документа

Нормативные правила партии. Если правила нет здесь (или в Combat) — его нет в игре.

## 2. Утверждённые правила (MVP)

### Старт

- Игроков: **2**
- Первый ход: **случайный**
- Поле: **8×8**

### Очередь ходов

- Как в шахматах: **одно действие** выбранной фигурой → ход переходит сопернику
- Действие: перемещение / атака / способность (когда способность доступна фигуре)
- Пропуск хода разрешён (UI: «Пропустить ход»)
- Сдача завершает матч поражением сдающегося

### Движение (база)

- Базовый ход: **ортогонально** (↑ ↓ ← →) на **1** клетку
- Диагональный базовый ход **не** утверждён как база

### Победа / поражение

- Победа: уничтожены все **командиры (короли)** противника **или** сдача противника
- В MVP у одной из временных фракций может быть 2 командира (Cosmodesant) — legacy каталога
- В целевом мире: один Командир на армию, если не решено иное (см. Commander)

### Режимы (статус)

| Режим | Статус |
|-------|--------|
| Классический → Локальная (2 вкладки) | Играбелен |
| Классический → Онлайн | Заглушка |
| Спец / Кастом / ИИ / Песочница | Заглушки |
