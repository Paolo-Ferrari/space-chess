## 1. Цель документа

Зафиксировать, **зачем** существует CyberChess: Overclock Protocol, для кого, какой опыт обещаем и что сознательно **не** делаем. Все фичи и roadmap сверяются с этим текстом.

## 2. Разделы

### Elevator pitch

Пошаговая тактическая PvP на поле Night City: боевые сети, здоровье, атака, способности и стоимость — ощущение шахмат, которые «научились стрелять».

### Бренд и тон

- Бренд: **CyberChess: Overclock Protocol** (коротко: **Overclock**)
- Слоган: *Киберпанк-шахматы. Армии. Импланты. Перегрев.*
- Визуальный тон: Cyberpunk + Tactical Chess + Futuristic Military Interface (см. [BrandIdentity.md](../03_ux/BrandIdentity.md))

### Целевая аудитория

- Игроки пошаговой тактики и «умных» PvP
- Фанаты киберпанк-эстетики Night City
- Команда разработки и стейкхолдеры (онбординг)

### USP (утверждённое направление)

- Фигуры как **роли / классы**, а не обязательный канон 1:1
- Фракции Night City + нейтральные эджраннеры
- Командир с имплантами; обязательный риппердок фракции
- Локальный PvP сейчас; онлайн — целевой этап (см. Roadmap)

### Non-goals (сейчас)

- Не симулятор открытого мира
- Не real-time action
- Не полный канон CDPR как лицензионный продукт (эстетика / вдохновение; юридические риски — Risk Register)
- Не выдумывать механики и баланс вне docs

### Критерии успеха (продуктовые, качественные)

- Новый разработчик понимает продукт за один день чтения docs
- MVP даёт завершённый матч по утверждённым правилам
- Целевой каталог фракций внедряется без ломки core loop

## 3. Что сюда записываем

Только утверждённое видение. Без таблиц юнитов и формул боя.

## 4. Когда используем

Старт проекта, питчи, онбординг, проверка scope любой крупной фичи.

## 5. Зависимости

- Входящие: решения владельца продукта
- Исходящие: GDD, Roadmap, UI Flow, Risk Register

## 6. Режим обновления

Редко — при смене направления продукта.

## 7. Открытые вопросы

- Позиционирование относительно других tactical PvP (когда появятся конкурентные референсы)
- Юридическая рамка использования имён/эстетики (Risk Register)
