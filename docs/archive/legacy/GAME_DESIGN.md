## Жанр и тон

Пошаговая тактическая PvP в духе шахмат на поле Night City: армии, здоровье, атака, способности, стоимость.

Игроков: **2**. Реального времени нет.

Слоган: *Шахматы умерли в XXI веке. В XXII они научились стрелять.*

Каждая фигура — прежде всего **роль / класс**, а не обязательный «конкретный человек из канона». Легендарные имена (Адам Смэшер, Джонни и т.д.) подчёркивают идентичность фракции или пул наёмников; это может быть «прототип класса», а не буквальная личность. Так мир можно расширять десятилетиями без ломки базовой механики.

# A. Что играбельно сейчас (MVP в коде)

Пока в билде: фракции-заглушки **Космодесант** / **Плутоны** + нейтралы. Целевой каталог Night City — ниже; замена каталога — отдельный этап.

### Победа и ход (уже в коде)

- Победа: уничтожены все **командиры (короли)** противника **или** сдача.
- Первый ход — случайный.
- Очередь как в шахматах: одно действие фигурой → ход сопернику.
- Базовый ход: ↑ ↓ ← → на 1 клетку.
- Поле **8×8**; локальная игра в 2 вкладках; UI Night City.

### Армия (уже в коде)

- Бюджет **100**, до **16** слотов; командиры обязательны.
- Сборки хранятся по герою/фракции.

# C. Эджраннеры (нейтральные наёмники)

**Не фракция.** Нейтральный пул легендарных / наёмных фигур, которыми можно усилить **любую** армию.

Примеры направления (имена = идентичность роли, не обязательно канон 1:1):

- Джонни Сильверхенд, Бестия, Люси, Ривер, Панам, Джуди, Такэмура;
- безымянные Соло / нетраннеры / техники.

Ограничение (дизайн): **не больше 2 эджраннеров на армию**, иначе сборки схлопнутся.

Примеры сборок: Militech + Люси + Панам; Arasaka + Бестия.

Полный список и стоимости — позже, отдельным ТЗ.

---

# D. Командир (вместо «просто короля»)

Командир — ключевая фигура армии.

- **Только Командир** получает **импланты** (выбор стиля: напр. Сандевистан / Берсерк / Кибердека — перечень и правила позже).
- Остальные фигуры фракции — **фиксированные** (без дерева имплантов на каждой).

Зачем: не балансить сотни комбинаций на всех юнитах; партии различаются через командира + эджраннеров + фракцию.

---

# E. Риппердок (обязательная фигура)

У каждой армии есть **свой риппердок**. Работает по-разному в зависимости от фракции (дизайн-направление):

| Фракция | Риппердок (идея) |
|---------|------------------|
| Arasaka | Защитные импланты, броня |
| Militech | Чинит мехов |
| Maelstrom | Не лечит: перегрузка — урон себе, усиление |
| Voodoo Boys | Снимает дебаффы; временная «перепрошивка» способностей |
| Tyger Claws | Ускоряет союзников |
| Valentinos | Лечит нескольких рядом |
| Nomads | Универсал: частичный ремонт техники + лечение людей |
| NCPD / MaxTac | Щиты |
| Animals | Сильное лечение, только одна цель |
| 6th Street | *(уточнить при внедрении каталога)* |

**Эджраннер-риппер** (отдельная идея): не принадлежит фракции; перед матчем выбирается специализация (хирург / техник / нетраннер / фармаколог) — один слот, разные стили.

Точные эффекты, дальность, кулдауны — **не придумывать** до отдельного ТЗ по цифрам.

# F. Победа и оформление финала

Базовое условие победы остаётся: **вывести из строя Командира** (сейчас в MVP — уничтожение королей/командиров; у целевого мира — один Командир на армию, если не решено иное).

Тематика финала может отличаться по фракции-победителю **без смены правила** (визуал / текст):

- Arasaka — физическое устранение;
- Voodoo Boys — взлом киберсистем;
- NCPD / MaxTac — полный контроль / «арест»;
- остальные — по духу фракции (уточнять при внедрении).

Сдача соперника по-прежнему завершает матч.
