## 0.2.0 — полировка и актуализация

### Игра

- Поле **16×16**; спавн армий с **4-й клетки** (файл `d`).
- Режим **Классический** без Hot Seat в названии: выбор **Онлайн** (заглушка) / **Оффлайн** (ходы по очереди).
- Оффлайн-цикл: подготовка → матч → победа → реванш / меню.
- UI поля: координаты `a–p` / `α–π`, глифы фигур, ATK/HP, карточка при наведении.
- Бренд **Space Chess** (меню, favicon, package name).

### Документация и репозиторий

- Docs приведены к фактическому состоянию (16×16, Классический Онлайн/Оффлайн).
- Удалены неиспользуемые стартовые файлы Vite (`App.css`, `public/icons.svg`).
- Добавлен корневой `README.md`.

---

## Каталог фракций (замена ростера)

- Удалены старые герои «Сигнал» / «Вектор» и связанные фигуры.
- Фракции: **Космодесант** (2 командира + уникальные) и **Плутоны** (герой + уникальные).
- **10 нейтральных** фигур для любой армии.
- Бюджет **100**, слоты **16**; несколько обязательных командиров.

---

## Классический режим (играбельный цикл)

- Подготовка (герой + армия) → матч 16×16 → ходы по очереди → победа по командирам.
- Активные способности: лечение, телепорт Архиона (раз в 5 ходов).

## 0.1.0 — новая концепция (старт)

Проект перезапущен под **пошаговую тактическую PvP**.

- Главное меню: Играть / Армии / Настройки.
- Экран **Армии**, домен отдельно от UI.
- Документация переписана под тактику (старая RTS как ТЗ снята).
