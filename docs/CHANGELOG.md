## 0.3.0 — правила, локальные вкладки, Night City

### Правила

- Случайный первый ход; победа по королям или сдаче.
- Очередь как в шахматах: одно действие фигурой → ход сопернику.
- Базовый ход: ↑ ↓ ← → на 1 клетку.
- Счётчик «Ход N» (не «раунд»).

### Локальная игра

- Классический → Локальная (2 вкладки): текущая = Игрок 1, вторая = Игрок 2.
- Общее состояние без сети; разворот поля; «Скрыть поле».

### UI

- Стиль Cyberpunk 2077 / Night City.
- Слоган в главном меню.

### Поле

- Доска **8×8**.
