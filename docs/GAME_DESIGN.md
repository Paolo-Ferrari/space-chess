# Space Chess — пошаговая тактическая PvP

Версия документа: 0.2.0

---

## Назначение

Единственный источник правды по текущей игровой концепции.

Если механики нет в этом файле — её ещё нет в игре.

Не придумывать механики сверх документа. Если неясно — спросить.

---

## Жанр

Пошаговая тактическая PvP-игра (**космические шахматы**).

Игроков: **2**.

Матч идёт строго по ходам. Реального времени нет.

---

## Цель матча

У каждого игрока есть **командир(ы)** фракции.

Победа — уничтожить всех командиров противника.

У **Космодесанта** обязательны два командира: оба должны быть уничтожены.

---

## Поле

Квадратная сетка клеток.

**Текущий режим (Классический):** карта **16×16**.

Координаты в UI: файлы `a–p`, ранги греческими буквами `α–π` (без арабских цифр на клетках).

Спавн армий: не с края поля, а с **4-й клетки** (файл `d`). Игрок 0 — нижние ряды, игрок 1 — верхние; заполнение слева направо, передний ряд первым.

На карте позже могут быть препятствия, стены, укрытия, особые клетки, биомы.

Сейчас — пустая сетка без сложных тайлов.

---

## Армия

Перед матчем игрок собирает армию в разделе **Армии** (как колоду).

Состав:

- **нейтральные юниты** — доступны всем;
- **уникальные юниты** — привязаны к фракции героя.

Лимиты: бюджет **100**, максимум **16** фигур в слотах.

У каждой фигуры: название, раса, стоимость, атака, здоровье, способность, краткое назначение.

В UI на поле: глиф фигуры, ATK/HP; при наведении — карточка юнита.

### Фракции (текущий каталог)

**Космодесант** — элитная броня и удержание. Два обязательных командира; перед матчем выбирается активный, после его гибели управление переходит ко второму. Победа — только после уничтожения обоих.

**Плутоны** — высокие технологии, мобильность и контроль. Особенность героя: раз в 5 ходов телепорт на свободную клетку рядом со своей фигурой (не под атакой врага).

Способности описаны в каталоге (технологии / биология / вооружение), без «магии».

---

## Стиль

- TRON / неон / киберпространство;
- минимализм;
- цифровые объекты.

Без фэнтези как сеттинга (шахматные глифы — только символы UI).

Текущий UI: тёмный фон, неоновый cyan/красный акцент, «стеклянные» панели.

---

## Геймплей хода

Игрок:

1. выбирает своего юнита;
2. выбирает действие (ход / атака / активная способность);
3. подтверждает;
4. завершает ход.

После окончания хода — ход противника (как в шахматах).

Перемещение и атака: соседняя клетка (включая диагональ). Один ход и одна атака на фигуру за ход игрока.

Активные способности сейчас: лечение (часть юнитов), телепорт Архиона (Плутоны, раз в 5 ходов).

---

## Режимы игры

В лобби **Играть**:

| Режим        | Статус                                      |
|--------------|---------------------------------------------|
| Классический | Играбелен: выбор Онлайн / Оффлайн           |
| Спец-режим   | Заглушка                                    |
| Кастомный    | Заглушка                                    |
| Против ИИ    | Заглушка                                    |
| Песочница    | Заглушка                                    |

**Классический → Онлайн:** сообщение «режим пока не готов» (сети нет).

**Классический → Оффлайн:** подготовка (герой + сохранённая армия для каждого) → матч 16×16 → ходы по очереди → победа по командирам.

Вне текущего скоупа: экономика, строительство, исследования, сеть, ИИ.

---

## UI (текущее)

Главное меню:

- Играть
- Армии
- Настройки

**Армии** — создание и редактирование (фракция + до 16 фигур, обязательные командиры). Бюджет **100**.

**Играть** — выбор режима и (для Классического) Онлайн/Оффлайн.

**Настройки** — заглушка.

Стиль: TRON / неон / тёмный терминал. Бренд: **Space Chess**.
